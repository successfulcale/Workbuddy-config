## 财务数据铁律（2026.06.11 松发股份教训）

背景：分析松发股份时，将2025年归母净利润2,655(百万元)=26.55亿，误读为2025年营收2,654亿元，凭空捏造了源文件不存在的数字。

从此以后，处理任何财务数据必须强制执行三校验：
1. 溯源标注——读到表格数据后先不分析，必须标注：原始单位、字段含义、来源位置。禁止凭表格行列位置猜测字段含义。
2. 正文交叉验证——表格数据必须与报告中文字描述的数字逐一核对。对不上就停下来，不强行解释。
3. 禁止凑数——不自行四舍五入、不填补缺失值、不从一个数字推导另一个再当作源数据引用。只能引用源文件实际存在的具体数字。

关键警戒：看起来合理的数据最危险。如果表格显示一个离奇的数字我会立刻警觉，但看起来合理的数据更容易让我跳过验证——这正是这次出错的根源。
