# 全量版报告结构

按以下模块顺序组织报告，各模块内子项根据实际情况增减。

---

## 报告开头：查询时间与股价信息

在"一、基本情况"模块之前，单独输出一段信息框：

```
报告时间: 2026-06-21 星期日 23:02 (非交易日,数据采用最近交易日收盘数据)
最新收盘价: 8.29元 (2026-06-18)  |  市盈率(TTM): -248.16
```

> **判断逻辑：** 获取系统当前时间，判断是否为交易日。非交易日（周末/节假日）则注明并采用最近交易日数据。

### （三）业绩预告

业绩预告分为**年报业绩预告**和**半年报业绩预告**两个独立判断流程。

**年报业绩预告逻辑：**
1. 最近完整年度年报是否已披露？是 → 跳过年报预告
2. 未披露 → 查询年报业绩预告 → 有则展示 | 无则跳过

**半年报业绩预告逻辑：**
1. 当年半年报（6月30日）是否已披露？是 → 跳过半年报预告
2. 未披露 → 判断当前日期是否在7月1日~7月15日之间
3. 在窗口期内 → 查询半年报业绩预告 → 有则展示 | 无则显示"未披露"
4. 不在窗口期内 → 跳过

**展示示例（年报预告）：**

```
年报业绩预告
| 项目 | 内容 |
|:---|:---|
| 报告期 | 2025-01-01 至 2025-12-31 |
| 预告类型 | 首亏 |
| 归母净利润（预计） | -2,500万元 至 -1,250万元 |
| 基本每股收益（预计） | -0.0431元/股 至 -0.0216元/股 |
| 上年同期归母净利润 | 2,815.34万元 |
| 数据来源 | 业绩预告公告 |
```

## 五、行业与竞争格局

- 所属行业规模及增速
- 产业链位置（上下游议价能力）
- 行业竞争格局（CR5/CR10、集中度变化）
- 可比公司对标（营收/利润/估值对比表）
- 行业政策环境及潜在影响

## 五、公告与资本运作

- 近年分红送转情况
- 股权激励/员工持股计划
- 再融资（定增/可转债/配股）
- 资产重组/并购
- 增减持（大股东/董监高）
- 回购进展

## 六、风险排查

### 6.1 法律与合规风险
- 未决诉讼/仲裁
- 行政处罚记录
- 监管问询函/关注函
- 立案调查

### 6.2 经营风险
- 客户/供应商集中度
- 商誉及减值风险
- 关联交易占比及公允性
- 大额担保/资金占用

### 6.3 财务风险
- 股权质押比例及平仓风险
- 债务到期及偿债能力
- 应收账款坏账风险
- 存货跌价风险

## 七、资金面与市场表现

- 近期股价走势（月线/季线）
- 成交量及换手率变化
- 北向资金/融资融券变化
- 主力资金流向

## 八、机构观点

- 近期研报覆盖情况（覆盖家数）
- 主流机构评级及目标价
- 一致预期（营收/净利润预测）
- 核心投资逻辑摘要

## 九、历史沿革（可选）

- 公司成立及改制过程
- 上市以来重大事件时间线
- 核心资产注入/剥离历史

## 十、综合评价与结论

- 核心优势（护城河）
- 主要风险点
- 估值判断（当前估值水平 vs 历史分位 vs 同业）
- 综合结论（正面/中性/谨慎）
